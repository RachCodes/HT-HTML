Formatting Guide

Tags and elements

Tags are key words used at the beginning of HTML elements to define how a browser will format and display content.They tell the browser what to expct and interpret the content between the tags.

HTML tags include three main parts:
  1. Opening tag <tag>
  2. Content
  3. Closing tag</tag>

Common HTML elements

There are many different elements used for a wide variety of purposes in HTML, but three common elements used concerning the page's metadata are as follows:

  • <html>
      • Defines the root eleent of the HTML document. This marks the beginning and end.
  • <title>
      • Assigns a title for the webpage, displayed in the title bar or page tab.
  • <body>
      • Identifies the body of the document, which contains all the visible content
A useful reference for elements and their uses can be found here.

Common errors

As W3 Schools point out, some elements will display correctly, even if you omit the end tag. However, they warn,

Never rely on this! Unexpected results and errors may occur if you forget the end tag!

Key Terms

  • Element A collection of a start tag, its attributes and values, an end tag and any content captured between the tags.
  • Tags Mark the start of an element, and define how the browser should format the content between the tags.
  • Attributes Proide supplementary information to the element to control its behaviour, for example in determining style. They have a value added to them to specify the precise formatting detail.
